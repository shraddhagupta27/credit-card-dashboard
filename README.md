# Credit Card Complaints Dashboard (Tableau)

### Project Overview
An interactive Tableau dashboard analyzing 87,000+ U.S. credit card complaints to assess company performance, response efficiency, and customer issue trends.

### Objectives
- Visualize complaint volumes and response trends over time  
- Track KPIs such as Timely Response Rate, Closed %, and Rolling 12-Month Complaints
- Identify high-complaint regions and recurring customer issues  
- Analyze submission channels and company responses for operational improvement  

### Tools & Techniques
- Tableau for dashboard design and interactivity  
- Calculated Fields for KPIs (timely response %, in-progress %, rolling metrics)  
- Filters & Parameters: Company, State, Year, Submission Channel   


### Skills Demonstrated
Tableau • Data Visualization • KPI Design • Financial Analytics • Customer Insights • Storytelling with Data
