# Credit Card Complaints Dashboard 
An interactive Tableau dashboard analyzing 87K+ U.S. consumer credit card complaints, with insights into complaint volume, root causes, company responses, and geographic trends.

## Overview

This project builds an end-to-end credit card complaints analytics dashboard in Tableau using Consumer Financial Protection Bureau (CFPB) complaint data. The dashboard provides a comprehensive view of complaint patterns across companies, states, issue types, and submission channels. It enables regulators, analysts, and financial institutions to identify systemic issues, track consumer sentiment, and evaluate operational response performance.
The solution includes time-series trend analysis, geographic heatmaps, top complaint categories, company response profiling, and daily/weekly complaint patterns.

## Problem Statement

Financial institutions receive thousands of complaints related to billing errors, fraud, customer service, late fees, and other credit-related issues.
Key challenges include: Lack of visibility into which complaints are increasing over time, No clear understanding of root causes and high-risk issue categories, Difficulty assessing company response effectiveness, Limited insights into state-wise and regional complaint trends
This Tableau dashboard solves these challenges by centralizing and visualizing complaint data to uncover actionable insights.
